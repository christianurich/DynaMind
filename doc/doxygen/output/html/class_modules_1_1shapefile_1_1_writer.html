def poly (   self,

  shapeType = POLYGON,
  partTypes = []

Creates a shape that has multiple collections of points (parts) including lines, polygons, and even multipoint shapes.

If no shape type is specified it defaults to 'polygon'. If no part types are specified (which they normally won't be) then all parts default to the shape type.

def record (   self,
  recordList,
  recordDict 
)

Creates a dbf attribute record.

You can submit either a sequence of field values or keyword arguments of field names and values. Before adding records you must add fields for the record values using the fields() method. If the record values exceed the number of fields the extra ones won't be added. In the case of using keyword arguments to specify field/value pairs only fields matching the already registered fields will be added.
